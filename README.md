# Modeling Metaphors: Analyzing Fine-Tuning Effects in ChatGPT
## A Reproducible Guide for Ethical AI Experiments
This project fine-tunes GPT-4o-mini using 20 metaphor-rich prompt-response examples related to AI ethics and business compliance. It analyzes how fine-tuning impacts metaphor use, framing, and clarity. It also serves as a reproducible teaching guide for students learning how to fine-tune LLMs to adopt tone, rhetorical strategies, and domain-specific voice.
